# 🧠 Heart Disease Risk Level Predictor

This is a machine learning–powered web app that predicts the risk level of heart disease based on user health data.

## 🔍 What it does
- Accepts user inputs like age, cholesterol, blood pressure, etc.
- Uses a trained regression model to predict risk
- Returns a risk score and risk level (Low, Moderate, High)

## 🚀 Tech Stack
- Python + Flask
- TailwindCSS for styling
- Scikit-learn (joblib) model
- HTML templates for frontend
